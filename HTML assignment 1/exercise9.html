HTML, which stands for Hyper Text Markup Language, is the standard markup language used to create and design web pages. It provides the structure for web content, allowing developers to define the layout, formatting, and functionality of a webpage.

Example Explained

In this example:

  • The DOCTYPE declaration specifies that this document is an HTML5 document.
  • The html element is the root element of the HTML page.
  • The head element contains meta-information about the HTML page, such as the title.

What is an HTML Element?

An HTML element consists of a start tag, content, and an end tag. It defines the structure and appearance of content on a web page. For example:

HTML Attributes

HTML attributes provide additional information about HTML elements. They are used to modify the behavior or appearance of an element.

HTML Headings

HTML headings are defined with tags from H1 to H6. They represent six levels of importance, with H1 being the most important and H6 being the least.
